# 택시운전 자격시험 랜덤 퀴즈

445개 문제 중 랜덤 70문항을 뽑아 풀 수 있는 정적 웹앱입니다. GitHub Pages에 그대로 올려 사용할 수 있습니다.

## 기능

- 445문제 중 랜덤 70문항 출제
- 문제 순서 및 보기 순서 랜덤 셔플
- 보기 클릭 즉시 정답/오답 표시
- 오답 선택 시 정답 보기 동시 강조
- 정답과 해설 즉시 표시
- 진행률, 정답/오답 개수, 실시간 점수 표시
- 70문항 중 42문항 이상 합격 기준 반영
- 중요 문제 저장 및 중요문제만 다시 풀기
- 완벽 문제 체크 및 숨기기 ON/OFF
- 오답노트 모드
- 시험 결과 화면
- 이어풀기 저장
- 문제 검색
- 이전 문제 다시 보기

## 실행 방법

브라우저에서 `index.html`을 열면 바로 실행됩니다.

## GitHub Pages 배포

1. 이 폴더의 파일을 GitHub 저장소에 업로드합니다.
2. 저장소 Settings > Pages에서 branch를 선택합니다.
3. 배포된 주소로 접속하면 퀴즈가 실행됩니다.
